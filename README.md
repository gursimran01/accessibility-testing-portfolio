# Accessibility Testing Portfolio

This repository showcases hands-on accessibility testing work performed using NVDA screen reader and keyboard-only navigation.

## What this portfolio includes
- NVDA screen reader testing on real public websites
- Page title, headings, landmarks, links, and keyboard navigation validation
- PDF accessibility testing including scanned vs text-based PDFs
- Documented findings with real observations and impact

## Tools Used
- NVDA Screen Reader
- Google Chrome
- Keyboard-only navigation
- Adobe Reader / PDF tools

## Repository Structure
- nvda-testing/ : Real NVDA accessibility test logs
- pdf-accessibility/ : PDF accessibility testing notes
- checklists/ : Reusable accessibility testing checklists

## Why this matters
Accessibility testing ensures digital products are usable by screen reader and keyboard-only users and supports WCAG / AODA compliance.

This portfolio reflects practical QA testing experience rather than theoretical certification.
