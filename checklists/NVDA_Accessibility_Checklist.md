# NVDA Accessibility Testing Checklist

## Page Information
- Page title announced correctly (NVDA + T)
- Title is descriptive and relevant

## Headings
- One H1 present
- Logical heading hierarchy
- No skipped heading levels

## Links
- Links are descriptive
- No "Click here" or ambiguous links

## Landmarks
- Banner, navigation, main, and footer landmarks present

## Keyboard Navigation
- Focus visible at all times
- No keyboard traps
- NVDA announces role and name correctly

## PDF Accessibility
- Text selectable
- NVDA reads content correctly
- OCR performed if scanned
